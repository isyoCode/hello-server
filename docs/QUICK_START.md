# 快速启动指南

## 🚀 30秒快速开始

### 1️⃣ 编译
```bash
cd /home/isyo/hello-git/net/main
bash build.sh
```

### 2️⃣ 启动
```bash
cd /home/isyo/hello-git
export LD_LIBRARY_PATH=/home/isyo/SqlPool/lib:$LD_LIBRARY_PATH
./net/main/hello-git-server
```

### 3️⃣ 打开浏览器
访问: **http://localhost:8888/auth.html**

### 注册新账号
在注册标签页中填写：
- 用户名：4-20字符（字母数字下划线）
- 邮箱：有效的邮箱格式
- 密码：8字符以上，包含大小写和数字

## 🧪 快速测试

### 测试打鱼游戏
1. 登录 → 进入游戏选择
2. 点击"开始游戏"（欢乐捕鱼）
3. 点击屏幕上的鱼（60秒倒计时）
4. 游戏结束后返回
5. 查看统计更新

### 测试搜索
1. 在搜索框输入 "鱼"
2. 应该只显示欢乐捕鱼

### 测试筛选
1. 点击 "简单" 按钮
2. 应该只显示欢乐捕鱼
